## Run Locally

This is a static website, so no build step is required.

1. Clone the repository.
2. Open example.html in a browser.

## Deployment

You can deploy this portfolio using GitHub Pages:

1. Go to repository Settings.
2. Open the Pages section.
3. Set source to the main branch (root).
4. Save and wait for the site URL to be generated.
